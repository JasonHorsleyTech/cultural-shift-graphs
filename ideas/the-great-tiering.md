# The Great Tiering

Spin-off from grandpas-eggs (angle-02).

When did product categories fragment into quality tiers? In 1955 there was one tier of eggs. Now there are six (conventional, cage-free, free-range, pasture-raised, organic, pasture-raised organic). Same pattern in coffee, college, airline seats, grocery stores themselves.

Questions:
- Is the proliferation of "good/better/best" a recent phenomenon or has it always existed?
- Which categories tiered first? Which resisted longest?
- Does tiering correlate with income inequality? (More tiers emerge as the income distribution widens?)
- Is tiering accelerating?

Could be a fascinating standalone graph — maybe a timeline showing when each tier appeared for a set of commodities.
