# Ticket 020: Meditation / MBSR — Evidence Assessment

## Objective

Determine whether there is sufficient QALY or cost-effectiveness data to include meditation/mindfulness-based stress reduction (MBSR) as a data point on the graph. This is a borderline intervention — include it if the evidence supports it, recommend exclusion if not.

## What to research

1. **Does any published CEA study report QALYs for meditation or MBSR?**
   - Search Tufts CEA Registry for "mindfulness" or "MBSR" or "meditation"
   - Check NICE evidence base (MBSR for recurrent depression is NICE-recommended)
   - PubMed: "mindfulness QALY" or "MBSR cost-effectiveness"

2. **If QALY data exists:**
   - What population was studied? (Clinical: cancer patients, chronic pain, depression relapse? Or general population?)
   - Does it match Bob's profile?
   - What was the QALY gain magnitude?
   - What was the cost of the program studied?

3. **If QALY data doesn't exist or is very thin:**
   - What outcome measures DO exist? (Depression scores, anxiety scores, cortisol, blood pressure)
   - Can these be reasonably converted to QALYs? (Probably not without significant hand-waving)
   - Recommend exclusion with explanation

4. **Cost of meditation for Bob:**
   - Free: guided meditation apps (Insight Timer), YouTube
   - Cheap: Headspace/Calm ($70/year)
   - MBSR course: typically $300-600 for 8-week program
   - Community meditation groups: free to donation-based

## Decision criteria

**Include** if: at least one published CEA study reports QALY gains for MBSR/meditation in a population remotely similar to general adults. Evidence tier would be 2-3.

**Exclude** if: QALY data only exists for narrow clinical populations (e.g., cancer patients) with no generalizability, or if no QALY data exists at all.

## Where to look

1. **Tufts CEA Registry**
2. **NICE** — mindfulness-based cognitive therapy (MBCT) for depression relapse
3. **Cochrane Reviews** — "Mindfulness-based interventions"
4. **PubMed** — "mindfulness QALY" or "meditation cost-effectiveness"

## Output format

If **include:** use the full intervention result format from plan.md.
If **exclude:** use the evidence assessment format (verdict + rationale + summary of what was found).
